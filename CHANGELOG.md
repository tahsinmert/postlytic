# Changelog

All notable changes to the **LinkedIn Post Analyzer** project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- **Viral Pattern Decoder**: New tool to analyze posts against PAS, AIDA, and Hero's Journey frameworks.
- **Hook & Fold Optimizer**: Real-time preview of how posts appear on mobile/desktop with "See More" fold detection.
- **Audience Persona Simulator**: AI-driven feedback simulating Recruiter, VC, and Founder perspectives.
- **Premium UI**: Complete overhaul of Profile, Settings, and Help pages with glassmorphism and ambient glow effects.
- **Project Documentation**: comprehensive README, Contributing guidelines, and Security policy.

### Changed
- Updated the `/tools` dashboard to feature dynamic, themed glass cards.
- Refactored the Comparison tool (`/compare`) for better visual hierarchy and animation.
- Migrated to a unified "Premium Glass" design system across the entire application.

### Fixed
- Improved mobile responsiveness on the settings page.
- Fixed layout shifts during authentication state loading.

## [0.1.0] - 2024-01-20
### Initial Release
- Basic post analysis functionality.
- User authentication with Firebase.
- Simple dashboard interface.
