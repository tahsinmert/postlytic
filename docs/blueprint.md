# **App Name**: Linkalyze

## Core Features:

- Post Analysis: Analyze raw LinkedIn post text to provide insights on hook effectiveness, structure, CTA, hashtags, and emoji usage, using AI tool.
- Virality Score: Calculate a virality score (0-100) for the post with a brief explanation of the score's factors (hook strength, structure, CTA clarity, hashtag usage, and engagement signals).
- Post Rewriting: Generate two rewritten post variants (professional and casual) that preserve the original meaning and intent of the input post, using AI tool.
- Dashboard: Display recent analysis and provide navigation to initiate new analyses and adjust user settings.
- Public Result Page: Create a shareable public page displaying the analysis results for a given post.
- User Authentication: Allow users to sign in with their Google accounts to access the application.
- Theme Preference: Allow the user to select their theme preference (dark vs light)

## Style Guidelines:

- Primary color: Neon purple (#A78BFA) to give a modern and premium feel.
- Background color: Dark grey (#1E293B) for a dark-first design.
- Accent color: Electric blue (#38BDF8) to highlight interactive elements.
- Body font: 'Inter', sans-serif, for a modern and neutral look, for both headlines and body text.
- Use 'lucide-react' icons for a consistent and clean aesthetic.
- Implement glassmorphism cards with a subtle blur effect for a modern look.
- Use subtle micro-animations using 'framer-motion' to enhance user experience.
