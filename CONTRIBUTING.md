## ğŸ”¨ Development Workflow

*   **Linting**: Run `npm run lint` before committing to ensure code quality.

*   **Commits**: We follow the **Conventional Commits** specification.

    *   `fix: resolve login redirect issue`

##  Pull Requests

1.  Fill out the pull request template completely.
2.  Include screenshots or videos for UI changes.
3.  Ensure all checks pass (linting, typechecking).
4.  Request a review from a maintainer.
